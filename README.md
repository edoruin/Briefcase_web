# Briefcase | Clases de Piano

Sitio web oficial de Briefcase, una plataforma de enseñanza de piano que combina disciplina profesional y alma artística.

## 🚀 Despliegue con GitHub Pages

Este proyecto utiliza **GitHub Actions** para automatizar el despliegue. Para activar el sitio:

1. Ve a los **Settings** de tu repositorio en GitHub.
2. En el menú lateral izquierdo, selecciona **Pages**.
3. En la sección **Build and deployment** $\rightarrow$ **Source**, cambia la opción a **GitHub Actions**.
4. Haz un `push` a la rama `main` y el workflow de despliegue se iniciará automáticamente.

## 🛠️ Estructura del Proyecto

- `index.html`: Página principal construida con Tailwind CSS.
- `logo-briefcase.jpg`: Activo visual de la marca.
- `.github/workflows/deploy.yml`: Configuración de automatización para el despliegue.

## 🎨 Diseño

El sitio sigue el sistema de diseño "Ivory & Ink", caracterizado por un minimalismo artístico con una paleta de colores basada en tonos tierra, obsidian y papel.
